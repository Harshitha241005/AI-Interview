FILE: src/hooks/useInterviewSession.js
------------------------------------------------------------------------------
PROMPT:
Create a custom React hook called useInterviewSession for managing an AI interview session.
Requirements:
- State: messages (array), isLoading, error, sessionConfig, isSessionActive, isStreaming
- startSession({ company, interviewType, difficulty, candidateName, apiKey }):
  * Builds system prompt using buildInterviewerSystemPrompt
  * Makes a POST to https://api.anthropic.com/v1/messages
  * Headers: Content-Type, x-api-key, anthropic-version: 2023-06-01, anthropic-dangerous-direct-browser-access: true
  * Model: claude-sonnet-4-20250514, max_tokens: 1024
  * First user message: "I'm ready to begin the interview." (mark as hidden: true)
  * Sets isSessionActive = true on success
- sendMessage(userInput):
  * Appends user message to messages state
  * Uses streaming (stream: true) with SSE parsing
  * Updates assistant message in real-time as chunks arrive (streaming: true flag on message)
  * Uses AbortController for cancellation
  * Full conversation history sent each turn
- endSession(): aborts any ongoing stream, sets isSessionActive = false
- resetSession(): clears all state back to initial
Return all state and functions.
Output: complete hook using useCallback, useRef, useState.
